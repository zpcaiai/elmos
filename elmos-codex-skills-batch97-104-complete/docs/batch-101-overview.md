# Batch 101 — Three Golden Route Packs

## Purpose

Implements three representative repository-to-PR routes that prove the platform on real technology estates.

System objective: deliver real Java, .NET and Python modernization routes from repository connection and baseline through deterministic transformation, constrained repair, verification, PR, rollback and route certification.
