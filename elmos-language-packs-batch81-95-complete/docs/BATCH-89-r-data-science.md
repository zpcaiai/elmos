## Safety boundary

Statistical conclusions must disclose data coverage, assumptions, random seeds, packages, and uncertainty; generated analyses are not automatically scientifically valid.

## Principal risks

- statistical method drift
- random-seed instability
- package version drift
- locale/timezone effect
- data leakage
- interactive session dependency
