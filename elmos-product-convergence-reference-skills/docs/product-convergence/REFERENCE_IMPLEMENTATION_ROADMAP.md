
# Reference Implementation Roadmap

## Phase 1：产品逻辑收敛
统一元模型、依赖图、全局状态机、Workflow、Evidence、Policy和Skill Registry。

## Phase 2：最小产品内核
实现模块化Control Plane和安全Private Runner。

## Phase 3：Reference Route
实现Java 17/21 + Spring Boot → C# + ASP.NET Core + EF Core。

## Phase 4：验证实验室
完成双运行、行为差分、Golden、性能、安全和Maintainability Gate。

## Phase 5：Design Partner
完成两个独立客户的P0切片、Canary、接管和验收。

## Phase 6：商业复制
冻结三类Edition、五类套餐、VMW指标和单位经济模型。
