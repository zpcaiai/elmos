# Batch 32 client modernization matrix

Each direction is independently certified. Example families include:

- JSP, JSF, or Thymeleaf to an exact React, Angular, Vue, or other modern target.
- Razor, ASP.NET MVC, or Web Forms to a modern web target.
- AngularJS to modern Angular, React, or Vue.
- JavaScript to TypeScript strengthening within an existing framework.
- WPF, WinForms, Swing, JavaFX, or native desktop to web, cross-platform desktop, or a hybrid shell.
- Native or hybrid mobile to native or cross-platform mobile.

Each tuple records exact versions and providers for source and target frameworks, languages, runtimes, build tools, package managers, routers, renderers, state, forms, styling, design system, API client, identity, i18n, tests, browsers, devices, and deployment.

Capability states:

- `certified`
- `supported`
- `conditional`
- `experimental`
- `detected-only`
- `blocked`

A framework name never implies all provider combinations. React with one router, renderer, state, form, CSS, identity, test, and browser profile is a different evidence tuple from another combination.
