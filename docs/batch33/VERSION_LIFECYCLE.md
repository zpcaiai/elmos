
# Version and lifecycle policy

- A provider family, IaC tool family, Kubernetes family, or CI provider is not a certification tuple.
- Lock provider plugins, modules, images, runners, actions, charts, policies, and CLI versions.
- Record state schema, workflow, and provider API compatibility.
- Treat source and target region, account model, identity, and managed-service tier as part of the tuple.
- Recertify for provider major versions, Kubernetes minor changes, module major changes, policy changes, state-backend changes, security-boundary changes, and material managed-service changes.
- Support research, experimental, limited, certified, deprecated, and blocked pack states.
