# Batch 34 shared contracts

Read all documents in this directory before implementing a portfolio-scale skill.

The executable local product slice lives in `modules/portfolio-scale`. Run `make batch34-local-rehearsal` to discover the current Maven reactor, materialize `portfolio-packs/elmos-local-rehearsal`, validate its typed inventory, graph, and work units, and invoke the conservative Batch 34 gate. This command intentionally returns `NOT_CERTIFIED` unless real independent scale and production evidence exists.
