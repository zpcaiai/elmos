# Batch 38–45 strictness profile

Certification requires 100% P0, 100% P1, at least 98% P2, at least 99% evidence traceability, complete independent holdout and representative evidence, all eight M38–M45 domain gates certified, two distinct design-partner organizations, one independent review, zero stale evidence, and zero zero-tolerance findings. Performance cases additionally enforce at most 5% P95 latency, 10% P99 latency, and 8% unit-cost regression.

P0 and P1 cannot be waived. A P2 waiver is not a pass, must be independently approved and expiring, and still cannot conceal a zero-tolerance finding. Missing execution or authorization remains `not-run` / `NOT_RUN` and blocks certification.
