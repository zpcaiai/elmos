# Slightly Strict Profile

Release requires 100% P0 and P1 pass, 100% holdout and representative workloads for certification-required scopes, >=98% evidence and source-target traceability, >=97% affected-test recall, >=85% mutation score, <=5% P95 and <=10% P99 latency regression, zero critical security/tenant/data/test-integrity issues, zero stale evidence and zero unreplayed critical failures.
