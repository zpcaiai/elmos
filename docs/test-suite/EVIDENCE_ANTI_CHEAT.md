# Evidence and Anti-cheating

Forbidden: editing certification status, deleting failed logs, auto-updating Golden from target output, widening masks/tolerances after failure, disabling security/validation/tests, hiding inaccessible repositories, using mocks as proof of real runtime, self-approval, reusing stale evidence, recording summaries without raw immutable evidence, keeping the trust anchor inside the evidence bundle, reusing artifact/environment placeholders, or signing a request that does not bind all 408 exact result and evidence-manifest digests.
