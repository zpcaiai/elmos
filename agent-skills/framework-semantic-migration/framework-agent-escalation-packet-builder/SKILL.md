# Framework Agent Escalation Packet Builder
Read `../references/afsm-v1.md`. Include only related AFSM/UIR entities, source/config fragments, target skeleton patterns, recipes, diagnostics, obligations and exact validation commands.

Lock routes, public DTOs, security and transaction contracts; restrict dependencies and patch boundaries. Accept only structured patches, candidate recipes, tests or unresolved reports. Reparse and run isolated validation, require human review for security/transaction/payment/permission output and cap retries.
