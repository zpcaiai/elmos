# Target Framework Code Emitter
Read `../references/afsm-v1.md`. Execute the recipe plan without making migration decisions. Use target-native syntax APIs for bootstrap, modules, endpoints, pipeline, providers, validation, errors, security, configuration, persistence, messaging, cache, scheduling and health.

Reparse, register, format and source-map every artifact. Enforce target version and dependency policy, deterministic ordering and non-placeholder output. Return `BLOCKED` when a native backend or binding proof is missing; do not fall back to large strings.
