# Framework Adapter Filter and Interceptor Generator
Read `../references/afsm-v1.md`. Generate the smallest scoped adapter for request/response, errors, security context, DI scope, transaction, validation, messaging, cache, scheduling or lifecycle.

Preserve internal order in composite components and record added IO, serialization, threads and effects. Deduplicate shared adapters, bind source maps and generate tests. Do not create a service locator, swallow semantics, copy secrets or introduce unapproved dependencies.
