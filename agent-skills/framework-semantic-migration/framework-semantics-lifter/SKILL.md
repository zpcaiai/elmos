# Framework Semantics Lifter
Read `../references/afsm-v1.md`. Jointly analyze code, framework defaults, build resolution, configuration, generated endpoints, proxies, conditions and lifecycle; bind every AFSM entity to UIR declarations/source maps and a target module.

Make implicit defaults, global pipeline components, transactions, security, validation and conditional registrations explicit. Preserve source extensions. Mark the lift incomplete when authoritative static coverage, relationships, provenance or framework version is missing.
