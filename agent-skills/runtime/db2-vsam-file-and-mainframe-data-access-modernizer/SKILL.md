# Mainframe Data Access Modernization

## Inventory and correlate

1. Link COBOL SQL statements through DBRM, package, collection, plan, and observed workload.
2. Preserve isolation, release, validate, current-data, degree, reopt, qualifier, owner, and authorization bind semantics.
3. Inventory KSDS, ESDS, RRDS, LDS, alternate indexes, keys, record layouts, sharing, recovery, and access patterns.
4. Record DSORG, RECFM, LRECL, BLKSIZE, encoding, copybook, sort order, trailers, control totals, generations, and retention for files.
5. Combine copybook layout, Db2 or VSAM definition, and runtime profile into a versioned data contract.

## Migrate safely

- Treat CDC as replication, not an automatic transfer of source authority.
- Verify alternate-index uniqueness and duplicate-key behavior.
- Reconcile keys, counts, control totals, amounts, statuses, deletes, duplicates, and business invariants.
- Keep application, data replication, and writer cutover as independently approved phases.
