## TMO职责

Strategy Integration
Portfolio Integration
Dependency Management
Change Portfolio
Decision Support
Transformation Health
Benefit Governance
Executive Communication
Assurance
Capability Building

## TMO不负责

- 替业务Owner实现收益；
- 替Product Owner管理产品；
- 替项目团队执行所有任务；
- 审批每项小变更；
- 制造重复报告；
- 永久拥有交付团队。

## Assurance

- Strategic Alignment；
- Governance；
- Benefits；
- Dependencies；
- Change Impact；
- Readiness；
- Controls；
- Closure。

## 验收标准

- TMO拥有明确服务；
- 论坛有Decision Rights；
- Cadence按需要；
- 报告只为决策服务；
- 业务Owner保留责任；
- Assurance与Delivery分开；
- TMO价值可衡量。
