## Sunk Cost

历史支出保留用于：

- TCO；
- Audit；
- Learning。

不能作为继续投资的必然理由。

## Exit

停止产品或Initiative时处理：

- People；
- Contract；
- Data；
- Users；
- Assets；
- Capitalized Software；
- Vendor；
- Archive；
- Security；
- Communication。

## 验收标准

- Funding定期重评；
- 条件可监控；
- Stop和Pivot为正常决策；
- 历史投入不主导；
- 退出有成本；
- 产品退出保护用户和数据；
- 资金释放进入Portfolio；
- Decision完整Audit。
