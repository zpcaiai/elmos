# Workflow, Rule, Report and Metric Modernization

## Execute

1. Inventory short transactions, long-running processes, human approvals, schedules, cases, sagas, and manual augmentation.
2. Locate validation, pricing, tax, credit, approval, routing, commission, accounting, assignment, and eligibility rules.
3. Separate suite configuration, custom code, Flow, plugin, trigger, report, integration, and external rule-service ownership.
4. Select keep-standard, reconfigure, domain service, rule engine, workflow, analytics, or retirement.
5. Preserve human task identity, limits, timers, escalation, compensation, audit, and in-flight state.
6. Version report source, query, metric, filter, security, schedule, export, consumer, and usage.
7. Define every metric by ID, owner, formula, grain, currency, timezone, and effective date.

## Gate

Validate report security separately from numeric equality. Treat unused reports as candidates until seasonal consumers and owners are resolved.
