## Sequencing

先完成：

- Identity；
- Data Ownership；
- Platform；
- Contract；
- Test；
- Observability；
- Migration Foundation。

再迁移依赖它们的应用。

## Dependency Loop

出现循环依赖时：

- 引入Facade；
- 分离Data Authority；
- 先建共享平台；
- 建立Temporary Adapter；
- 调整Capability Boundary。

## 验收标准

- Roadmap高于项目列表；
- Wave关联Target Gap；
- Dependency可执行；
- Enabler可见；
- Business Calendar纳入；
- Decision Gate明确；
- Legacy Decommission作为Milestone；
- Roadmap可根据Evidence重新计划。
