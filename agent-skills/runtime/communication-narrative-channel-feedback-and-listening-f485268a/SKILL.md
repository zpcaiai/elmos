## 验收标准

- 沟通是双向；
- Audience和Messenger明确；
- Why和Personal Impact分开；
- 发送不等于理解；
- Feedback进入Concern；
- 多语言和Accessibility支持；
- 承诺内容经过批准；
- 信息版本可追踪。
