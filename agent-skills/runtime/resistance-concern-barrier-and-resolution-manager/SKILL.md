## 禁止

- 建立个人“抵抗者名单”；
- 报复提出Concern的人；
- 把所有异议当成沟通失败；
- 关闭Concern而不记录理由。

## 验收标准

- Concern与个人标签分开；
- Barrier可操作；
- 有Evidence和Owner；
- 有效风险进入Portfolio；
- Resolution可验证；
- 拒绝有透明理由；
- 可重新打开；
- 聚合Concern反馈设计缺陷。
