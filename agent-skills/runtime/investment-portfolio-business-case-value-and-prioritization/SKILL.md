## TBM映射

成本和投资可选择映射到TBM Taxonomy Provider，用于把技术财务和运营数据连接到业务价值；ELMOS仍保留自身Portfolio对象。TBM Council当前将该实践描述为把技术投资与可衡量业务价值关联。

## 验收标准

- Initiative关联Capability；
- Build和长期Run Cost分开；
- Benefit有Baseline；
- 避免重复计算收益；
- Mandatory和Strategic分开；
- 依赖进入优先级；
- 取消条件明确；
- 完成项目后继续验证收益。
