# Target Stack Decision Engine
Read `../references/skeleton-v1.md`. Score native fit, source semantic fit, organization fit, build/test maturity, maintenance, license/security, offline/private deployment and performance. Record selected/alternatives/reasons/constraints/tradeoffs/confidence/approval for every dimension. Do not choose by popularity, invent versions, select ORM before data semantics, or continue through conflicting hard constraints.
