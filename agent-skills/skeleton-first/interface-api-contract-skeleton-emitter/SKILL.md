# Interface API Contract Skeleton Emitter
Read `../references/skeleton-v1.md`. Preserve names/order/types/defaults/variadics/returns/async/errors/nullability/effects/generics/visibility/serialization and versioning. Generate DTO, endpoint, handler, message, callback/subscription and external schema placeholders without domain behavior. External names never change with code convention; unexpressible defaults/errors/async contracts produce overloads/adapters or obligations and compatibility diffs.
