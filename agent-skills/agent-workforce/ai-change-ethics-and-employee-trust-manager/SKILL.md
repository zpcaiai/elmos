## Authoritative specification

## Description

治理AI采用中的透明度、公平、员工参与和组织信任。

## Trust Questions

* 哪些工作被监控？
* 哪些数据被使用？
* Agent是否评价员工？
* 员工能否质疑Agent结果？
* 人事决策是否有人类复核？
* AI错误由谁负责？
* 数据保留多久？

## Ethics Review

适用于：

* 招聘；
  -绩效；
  -晋升；
  -薪酬；
  -解雇；
  -健康；
  -监控；
  -客户权益；
  -高风险决策。

## Hard Rules

* 不进行隐蔽员工监控；
* 人事高影响决策必须有人类判断；
* 员工可申诉；
* 敏感属性使用需合法和必要；
* Agent建议不得变成未经审查事实；
* AI变革需员工参与；
* 伦理问题进入治理委员会。

## Acceptance Criteria

* AI使用透明；
* 员工申诉有效；
* 高影响决策有人类监督；
* 隐私边界明确；
* 员工信任可测；
* 伦理风险得到处理。
