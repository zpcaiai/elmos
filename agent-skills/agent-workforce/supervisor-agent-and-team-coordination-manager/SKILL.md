## Authoritative specification

## Description

让Supervisor Agent分配任务、汇总结果和协调Specialist Agent。

## Supervisor职责

* 目标分解；
  -任务分配；
  -依赖管理；
  -预算；
  -进度；
  -质量汇总；
  -冲突升级；
  -停止。

## Supervisor限制

* 不创建未登记Agent；
  -不扩大权限；
  -不修改公司目标；
  -不跳过Verifier；
  -不自行接受高风险结果。

## Hard Rules

* Supervisor也有Human Owner；
* 子Agent预算从Team Budget扣除；
* 任务分配记录；
* 失败子Agent不可无限替换；
* Supervisor结论需证据；
* 层级深度有限；
* Supervisor失效可人工接管。

## Acceptance Criteria

* 多Agent任务可协调；
* 依赖顺序正确；
* 成本和进度可见；
* 冲突可升级；
* Supervisor不是不可解释黑箱；
* 团队结果优于无协调基线。
