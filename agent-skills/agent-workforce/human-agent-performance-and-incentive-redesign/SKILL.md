## Authoritative specification

## Description

调整绩效和激励，避免员工与Agent之间形成错误竞争或指标操纵。

## Performance Focus

```text
Outcome
Judgment
Quality
Collaboration
Agent Supervision
Process Improvement
Learning
Risk Management
```

## 不应奖励

* Agent调用次数；
  -自动化比例本身；
  -生成内容数量；
  -隐藏例外；
  -减少人工审批数量；
  -绕过政策提高速度。

## Hard Rules

* Human仍对职责范围结果负责；
* Agent错误需考虑系统和管理责任；
* 员工报告Agent问题不应受罚；
* 自动化节省可转化更高价值工作；
* 销售Agent不能激励垃圾外联；
* 绩效需考虑安全和质量；
* 激励设计需持续评估副作用。

## Acceptance Criteria

* 激励与公司Outcome一致；
* 员工愿意报告风险；
* 无Agent指标游戏；
* 人类能力持续成长；
* 监督责任明确；
* AI采用与员工信任共存。
