## Authoritative specification

## Description

管理晋升标准、跨团队校准和人才组合评审。

## Promotion Criteria

```text
持续达到下一职级
影响范围已扩大
复杂度和自主性提高
能力和行为符合标准
结果可持续
```

## Talent Review

关注：

* 高绩效；
  -高潜力；
  -关键专家；
  -继任候选；
  -流失风险；
  -岗位错配；
  -发展需求。

## Hard Rules

* 晋升不能只奖励超时工作；
* Title不能替代实际Scope；
* 单一项目成功不足以证明持续表现；
* 校准需使用相同标准；
* 预算不能成为不透明拒绝理由；
* 晋升反馈必须具体；
* 人才评审不得成为秘密标签系统。

## Acceptance Criteria

* 晋升公平；
* 跨团队标准一致；
* 关键人才得到发展；
* 流失风险可识别；
* 人才决策有证据；
* 职级膨胀受控。
