## Authoritative specification

## Description

建立管理者选择、培训、支持、评估和问责体系。

## Manager Responsibilities

```text
Set Direction
Clarify Roles
Allocate Resources
Develop People
Give Feedback
Make Decisions
Manage Performance
Create Safety
Deliver Results
```

## Development

* 新经理训练；
  -教练；
  -同伴组；
  -360反馈；
  -经营训练；
  -招聘训练；
  -财务训练；
  -风险训练；
  -董事会沟通。

## Hard Rules

* 最优秀专家不自动成为经理；
* 管理角色需明确选择；
* Manager需接受绩效评估；
* 频繁离职团队需调查管理问题；
* Manager跨度需受控；
* 技术和管理路径平等；
* 领导力原则需与Operating Principles一致。

## Acceptance Criteria

* 管理者具备基本能力；
* 团队目标和反馈清晰；
* 人才发展改善；
* Manager质量可衡量；
* 新经理失败率下降；
* 管理能力支持公司扩张。
