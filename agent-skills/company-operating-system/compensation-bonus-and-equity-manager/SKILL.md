## Authoritative specification

## Description

设计基本薪酬、奖金、佣金、长期激励和股权体系。

## Compensation Elements

```text
Base Salary
Variable Bonus
Sales Commission
Equity
Benefits
Allowances
Retention
Sign-on
```

## Compensation Philosophy

需明确：

* 市场定位；
  -现金与股权比例；
  -地区策略；
  -岗位稀缺；
  -绩效关系；
  -内部公平；
  -透明程度。

## Equity

管理：

* Option或股份；
  -Grant；
  -Vesting；
  -Cliff；
  -Exercise；
  -Leaver；
  -Dilution；
  -Refresh；
  -税务；
  -董事会批准。

## Hard Rules

* 同岗薪酬差异需有合理原因；
* 佣金不能鼓励低质量收入；
* 销售激励需考虑毛利和回款；
* Equity Grant需正确审批；
* 薪酬数据严格保密；
* 薪酬体系需符合地区法律；
* 不能用模糊未来股权替代正式Grant。

## Acceptance Criteria

* 薪酬有市场竞争力；
* 内部公平可分析；
* 激励与公司目标一致；
* 股权记录准确；
* 佣金计算可审计；
* 成本进入财务模型。
