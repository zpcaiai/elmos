## Material Standard

每个议题明确：

```text

For Discussion
For Decision
```

## Hard Rules

* 材料应提前分发；
* 决策事项需清楚说明建议；
* 数据与管理报告一致；
* 坏消息不得隐藏；
* 材料不应过度冗长；
* 敏感材料访问受控；
* 材料版本不可覆盖。

## Acceptance Criteria

* 董事有足够时间准备；
* 会议聚焦重大事项；
* 决策材料清晰；
* 财务和风险透明；
* Board Pack可审计；
* 信息质量持续改善。
