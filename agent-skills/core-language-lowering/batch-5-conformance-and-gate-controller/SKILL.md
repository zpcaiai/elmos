# Batch 5 Conformance and Gate Controller
Read `../references/lowering-v1.md`. Compute generation and validation separately; report deterministic, verified-agent, manual, opaque and placeholder populations, source-map coverage, and type/evaluation/numeric/collection/exception/async/nullability fidelity. Apply L-A through L-D to every module independently with stricter profiles for finance, payment, authorization, transaction and concurrency code.
