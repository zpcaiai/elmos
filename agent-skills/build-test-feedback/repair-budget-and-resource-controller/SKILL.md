# Repair Budget and Resource Controller
Read `../references/batch-8-repair-loop.md`. Prioritize build blockers, public contracts, security, transactions, test compilation and required tests. Attribute each call and charge unknown usage conservatively.

Stop the exhausted cluster, retain the stable Snapshot, create an escalation packet and continue only independent work. Do not repeat an invalid patch, expand budget silently or report completion after exhaustion. Escalate exhausted P0 work to a named human.
