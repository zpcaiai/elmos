## Hard Rules

* Portal权限与平台权限一致；
* API Key默认隐藏；
* 示例不能包含Secret；
* 版本文档明确；
* Deprecated功能显著标记；
* Portal Search需跨文档和社区；
* 私有客户内容不能公开索引。

## Acceptance Criteria

* 开发者可快速找到入口；
* Portal支持自助学习；
* 文档、社区和Marketplace关联；
* 版本和权限正确；
* 搜索质量可衡量；
* Portal推动Activation。
