## Hard Rules

* 代码标识和API名称不翻译；
* 技术术语需与本地开发者习惯一致；
* 同一词在不同上下文可有不同翻译；
* 品牌词和产品名称遵守规则；
* 术语更新需影响检查；
* Translation Memory不能跨客户泄露私有内容；
* 法律文本单独维护。

## Acceptance Criteria

* 关键术语一致；
* 翻译效率提高；
* 技术误解减少；
* 多供应商输出一致；
* 术语变更可追踪；
* 地区语言习惯得到支持。
