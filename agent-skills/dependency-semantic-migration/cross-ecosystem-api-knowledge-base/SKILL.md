# Cross Ecosystem API Knowledge Base
Read `../references/dependency-migration-v1.md`. Store API records separately from mapping records. Version source/target ranges, API-level correspondences, semantic claims, known gaps, strategies, adapters, platform/license/support metadata, provenance, reviewer, timestamps, confidence and approval state. Retrieval may aid discovery but cannot grant equivalence. Never overwrite historical approvals, merge contradictory claims, or promote model-generated knowledge without authoritative evidence and validation. Feed differential failures back as new versioned counterexamples.
