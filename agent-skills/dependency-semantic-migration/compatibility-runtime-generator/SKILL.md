# Compatibility Runtime Generator
Read `../references/dependency-migration-v1.md`. Emit target modules, APIs, implementation, tests, package metadata, version manifest, observability hooks and consumer examples with deterministic generated-region ownership. Validate build, API surface and behavioral obligations through injected backends. Never generate an unversioned global utility bucket, hide target dependencies, execute package scripts, or claim compatibility from compilation alone. Unimplemented contract members and unverified semantic behavior remain blocking.
