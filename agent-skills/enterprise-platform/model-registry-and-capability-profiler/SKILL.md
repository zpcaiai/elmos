# Model Registry and Capability Profiler

Read `../references/batch-12-enterprise-platform.md`. Record immutable model/deployment IDs, evaluation status, context/tool/structured-output capability, policy classes, region, retention, runtime/tokenizer and license. Tenant policy may only narrow platform approval.

Display names are not identity. Unassessed, unlicensed or stale models cannot serve production migration and block T-D/T-F where required.
