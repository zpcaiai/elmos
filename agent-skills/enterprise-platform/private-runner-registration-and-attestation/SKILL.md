# Private Runner Registration and Attestation

Read `../references/batch-12-enterprise-platform.md`. Verify signed Runner version, host/architecture, sandbox, runtime, network policy, disk, time, certificate and toolchains before issuing identity. Bind the machine and pool immutably; quarantine stale or failed versions.

Reusable tokens, cloned identity, self-selected tenant/pool or missing attestation blocks T-C.
