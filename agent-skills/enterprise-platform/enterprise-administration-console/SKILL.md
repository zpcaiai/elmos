# Enterprise Administration Console

Read `../references/batch-12-enterprise-platform.md`. Use the same APIs and policies as automation; show role-scoped modules, mask sensitive values and require step-up, preview, diff, approval and auditable recovery for risky changes. Clearly mark support impersonation.

The console has no hidden privilege. Cross-tenant visibility or UI-only authorization blocks T-B/T-G.
