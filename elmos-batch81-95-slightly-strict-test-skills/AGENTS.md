# Test Agent Operating Rules

- Implementer and independent certifier must be separate roles for Critical assertions.
- Never edit source requirements, approved tolerances or source Skills to make a test pass.
- Preserve every failed attempt and evidence reference.
- Use vendor sandboxes, simulators and licensed runtimes only through approved capabilities.
- No production data, credentials, PLC outputs, SAP production clients or Salesforce production orgs.
- A Critical failure stops release certification immediately.
