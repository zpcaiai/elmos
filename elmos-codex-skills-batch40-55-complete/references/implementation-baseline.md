# Implementation baseline

Every Skill must:

1. reuse authoritative ELMOS aggregates;
2. preserve immutable source facts and versioned decisions;
3. enforce tenant and resource authorization;
4. use typed provider adapters and capability discovery;
5. preserve idempotency, reconciliation and recovery;
6. emit source-linked evidence and audit events;
7. fail closed for mandatory security, privacy, legal, financial or safety controls;
8. pass repository-native tests and the Skill-specific release gate.
